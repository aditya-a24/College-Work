Q12. A Program to implement CSS selectors in HTML



CSS Selectors



Type1. Universal Selector, every text of this web page is in sans-serif font, it is an example of Universal Selector.

Type2. ID Selector, this line is in Red Color, it is an example of ID Selector.

Type3. Class Selector, this line is in Green color, it is an example of Class Selector.

Type4. Element Selector, only word inside Span tag is Italic and Bigger, it is an example of Element Selector.

Type5. Group Selector, span of type4 and t3 of type3 class is in hover effect, it is an example of Group Selector.

Type6. Attribute Selector GEU WEBSITE hovering over Hyperlink, it is an example of Attribute Selector
